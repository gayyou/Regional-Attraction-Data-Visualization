# ICMK

### 项目路径

| 类别     | 具体名称 | 目录                |
| -------- | -------- | ------------------- |
| 组件存放 | 地图     | /components/map     |
|          | 浮出层   | /components/layer   |
|          | 侧边栏   | /components/panel   |
| 组成视图 | 主页面   | /views/index        |
| 工具     | 通用工具 | /utils/shared/utils |
|          | 模型对象 | /utils/shared/model |
| 静态资源 | 图片     | /assets/images      |
|          | 图标     | /assets/icons       |
|          | 字体     | /assets/fonts       |
